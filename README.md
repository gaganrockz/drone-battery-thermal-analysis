# drone-battery-thermal-analysis
Thermo-fluidic modeling of air-cooled LiPo battery pack for UAV operations.


## Project Structure

01-geometry/ - CAD models (.step, .stl files)
02-mesh/ - Mesh files and quality reports
03-simulation-files/ - CFD and MATLAB files
04-results/ - Simulation output data
05-report/ - Final report and screenshots
06-documentation/ - Assumptions, equations, contributions
